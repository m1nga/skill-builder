# 疑难杂症 & 技巧

## 结块 / 静电（粉发干、抱团、挂壁、喷溅）
**原因**：静电。浅烘豆致密、含水低，加上干燥环境，磨完特别容易带静电——粉抱团、喷得到处都是、残留在磨豆机里。结块还会让布粉不匀 → 通道效应 → 萃取忽高忽低。

**解法：RDT（Ross Droplet Technique，布水消静电法）**
- 磨豆**前**，用牙签或勺子尖蘸**一滴水**，搅一下豆子（或极轻喷一下雾）
- 一滴就够，别多——目的是消静电，不是泡豆
- 效果：几乎消灭结块和喷溅，残粉大减，布粉更匀，萃取更稳
- 浅烘 + 干燥季节基本是标配

## WDT（Weiss Distribution Technique，布粉消团法）
磨粉入碗后，用细针（专用布粉针或几根别针）轻轻搅散结块、把粉床拨平，再压粉。消除磨豆机带来的结块和密度不均，是防通道效应最有效的一招。

## 通道效应（channeling）
**现象**：水从粉饼薄弱处/裂缝走捷径 → 局部过萃 + 整体欠萃 → 浓缩四处喷射(squirt)、流速忽快、口感又酸又涩。
**原因**：布粉不匀、有结块、压粉歪、粉太满顶到分水网、粉碗边缘有空隙。
**解法**：
- RDT + WDT 布粉
- 压粉水平、力度一致
- 粉量别超过粉碗容量（顶到分水网会乱喷）→ 必要时减 1g
- 有条件用无底手柄(naked portafilter)观察萃取，喷射就是通道

## 萃取口感诊断（完整版）
| | 欠萃（萃取不足） | 过萃（萃取过度） |
|--|--|--|
| 口感 | 酸、尖、涩、咸、空、不甜、收尾干 | 苦、焦、木、呛、灰、回味发干发涩 |
| 常见原因 | 太粗、太快、太短、水温低、浅烘没吃透 | 太细、太慢、太长、水温高 |
| 怎么救 | 磨细、拉长比例/时间、提温 | 磨粗、缩短比例/时间、降温 |

**关键提醒**：两者入口都"不舒服"，最容易认错。**主要发酸 → 欠萃；主要发苦 → 过萃**。浅烘的欠萃尖酸常被误当成过萃，先看「时间 + 研磨」这两个客观数字再下判断。

## 新豆排气（养豆 <7 天）
豆子太新含大量 CO₂，意式易通道、手冲闷蒸狂涨泡。
- 意式：养够 7–10 天再认真调；急用就磨粗半格、稍降萃取
- 手冲：闷蒸水量加大、时间延长（35–45s）

## 浓缩 / crema 外观
- 浅烘：crema 薄、色浅，正常，别因此判断失败
- 一开始就发白、流速快 → 欠萃或通道
- 极慢、发黑、几乎无 crema → 过萃或磨太细堵住
